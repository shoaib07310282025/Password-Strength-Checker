#  Password Strength Checker
A simple Python-based Password Strength Checker that evaluates the strength of a password using multiple security criteria such as:
- Uppercase letters
- Lowercase letters
- Digits
- Special characters
- Password length
- Common password detection
This project was developed as part of the AICT Course Project at PIEAS.
